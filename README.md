# Semantic Search Engine with RDF & BERT

A Python-based **semantic search engine** that understands the meaning of queries and returns the most relevant documents from an RDF knowledge graph.

## Features
- Store documents in **RDF** format
- Query using **SPARQL** (optional filtering)
- Compute **BERT embeddings** for semantic similarity
- Fully interactive in **Jupyter Notebook**

## Example
Query: "Which laptops are best for AI development?"  
Top results:
- Best Laptops for AI (Similarity: 0.72)
- Top AI Tools and Hardware (Similarity: 0.58)
- Affordable Laptops for Students (Similarity: 0.46)
